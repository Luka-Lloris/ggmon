## 버전 설명

### [1.0.0] - 초기 릴리스 (2024-01-15)

**핵심 기능:**
- ✅ 프리랜서 계약서 자동 생성
- ✅ 법적 리스크 최소화 구조
- ✅ 3.3% 원천징수 정확한 계산
- ✅ JSON 스키마 기반 데이터 검증
- ✅ Git 기반 이력 관리

**포함된 템플릿:**
1. `freelancer_contract.md` - 프리랜서 용역계약서
   - 근로계약 아님 명시
   - 4대보험/퇴직금 제외 확인
   - 업무 자율성 보장
   
2. `daily_worker_contract.md` - 일용직 근로계약서
   - 근로기준법 준수
   - 일당 기반 계약
   
3. `nda.md` - 비밀유지계약서
   - 영업비밀 보호
   - 계약 종료 후에도 유효

**스키마 구조:**
- `worker.json` - 인력 정보 (ID, 이름, 연락처, 계좌 등)
- `contract.json` - 계약 정보 (기간, 업무, 보수 등)
- `payment.json` - 지급 정보 (금액, 세금, 상태 등)

**주요 스크립트:**
- `generate_contract.py` - 메인 생성기
  - 변수 자동 치환
  - 실수령액 계산
  - NDA 자동 생성
  
- `validator.py` - 데이터 검증기
  - 필수 필드 확인
  - 형식 검증
  - 비즈니스 로직 검증
  
- `pdf_converter.py` - PDF 변환기
  - 한글 폰트 지원
  - A4 레이아웃 최적화

**법적 안전장치:**
1. 근로자성 부인 조항
2. 업무 자율성 명시
3. 원천징수 자동 계산
4. Git 버전 관리로 변조 방지
5. 필수 필드 검증으로 누락 방지
